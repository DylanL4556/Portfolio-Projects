This is a project I've enjoyed a lot.

Involving scraping data from existing public password leaks, and annotating it to various degrees, creating a dataset of password pairs with a label which corresponds to
whether or not a password pair is from the same user, or two different users. What is included (which is not the entirety of the project) is various scripts and annotated databases created in 
our experiments, the files included are the most important amongst our repository but there were a few dozen scripts not included.

The included scripts range from parsing data, annotating data, running ML models, to creating graphical representations of important data.
